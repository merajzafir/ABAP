Sample Code: AT SELECTION-SCREEN OUTPUT 
REPORT z31_sel_screen.

PARAMETERS p_enable AS CHECKBOX USER-COMMAND UC1.
PARAMETERS: input(5) TYPE c MODIF ID mod .

AT SELECTION-SCREEN OUTPUT.
  LOOP AT SCREEN.
    IF p_enable = 'X'.
      IF screen-group1 = 'MOD' .
        screen-active = 1.

      ENDIF.
    ELSE.
      IF screen-group1 = 'MOD' .
        screen-active = 0.

      ENDIF.
    ENDIF.
    MODIFY SCREEN.

  ENDLOOP.
